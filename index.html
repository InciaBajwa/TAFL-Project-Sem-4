Core Definitions

DFA

A Deterministic Finite Automaton has exactly one transition for each symbol from each state. At any step, only one next state is possible.

NFA

A Nondeterministic Finite Automaton can have multiple possible transitions for the same symbol. At any step, many next states may be explored in parallel.
